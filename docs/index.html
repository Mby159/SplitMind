🧠 SplitMind

隐私优先的 AI 任务编排系统

在享受 AI 能力的同时，完全掌控你的数据隐私

快速开始 GitHub

核心特性

🔒

隐私保护

自动检测并脱敏敏感信息，所有隐私处理都在本地完成，确保数据不会暴露给任何 AI 提供商。

🧩

智能拆分

智能分析任务类型，自动拆分为独立子任务，支持多种拆分策略，适应不同场景需求。

🔀

多提供商支持

支持 OpenAI、Anthropic Claude、Kimi 等主流 AI 提供商，以及本地模型，避免供应商锁定。

⚡

灵活执行模式

三种执行模式（Local Only / Hybrid / Online），在隐私保护和 AI 能力之间自由平衡。

🔄

智能聚合

多种结果聚合策略，自动去重、补充和优化，确保最终输出质量。

💻

多种使用方式

提供 SDK、CLI 工具和 Web 界面，满足不同用户的使用习惯。

工作原理

1

隐私保护

在本地检测并脱敏敏感信息，生成隐私报告

2

任务拆分

智能分析任务，拆分为多个独立子任务

3

分布式执行

将脱敏后的子任务分配给不同 AI 提供商

4

结果聚合

在本地聚合结果，并还原敏感信息

快速开始

# 安装 SplitMind
pip install splitmind

# 零配置体验隐私保护
splitmind demo "你的文本"

# 使用 Python SDK
from splitmind import SplitMindEngine

engine = SplitMindEngine()
result = await engine.execute(
    task="分析这份报告",
    context=report_text,
    execution_mode="hybrid"  # 平衡隐私和能力
)

print(result.final_result)
